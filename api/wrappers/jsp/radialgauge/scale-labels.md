# \<kendo:radialGauge-scale-labels\>
A JSP tag representing Kendo Labels.

#### Example
    <kendo:radialGauge-scale>
        <kendo:radialGauge-scale-labels></kendo:radialGauge-scale-labels>
    </kendo:radialGauge-scale>


## Configuration Attributes


### background `String`

The background color of the labels.
Any valid CSS color string will work here, including hex and rgb

#### Example
    <kendo:radialGauge background="background">
    </kendo:radialGauge>



### color `String`

The text color of the labels.
Any valid CSS color string will work here, including hex and rgb.

#### Example
    <kendo:radialGauge color="color">
    </kendo:radialGauge>



### font `String`

The font style of the labels.

#### Example
    <kendo:radialGauge font="font">
    </kendo:radialGauge>



### format `String`

The format of the labels.

#### Example
    <kendo:radialGauge format="format">
    </kendo:radialGauge>



### margin `float`

The margin of the labels.

#### Example
    <kendo:radialGauge margin="margin">
    </kendo:radialGauge>



### padding `float`

The padding of the labels.

#### Example
    <kendo:radialGauge padding="padding">
    </kendo:radialGauge>



### position `String`

The labels positions.

#### Example
    <kendo:radialGauge position="position">
    </kendo:radialGauge>



### template `String`

The label template.
Template variables:

#### Example
    <kendo:radialGauge template="template">
    </kendo:radialGauge>



### visible `boolean`

The visibility of the labels.

#### Example
    <kendo:radialGauge visible="visible">
    </kendo:radialGauge>
