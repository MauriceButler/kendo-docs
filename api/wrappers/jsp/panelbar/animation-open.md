# <kendo:panelBar-animation-open>
A JSP tag representing Kendo Open.

## Configuration Attributes


### duration `float`

The number of milliseconds used for the visual animation when an item is opened.

#### Example
    <kendo:panelBar duration="duration">
    </kendo:panelBar>
