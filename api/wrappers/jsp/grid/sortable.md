# <kendo:grid-sortable>
A JSP tag representing Kendo Sortable.

## Configuration Attributes


### allowUnsort `boolean`

Defines whether column can have unsorted state.

#### Example
    <kendo:grid allowUnsort="allowUnsort">
    </kendo:grid>



### mode `String`

Defines sorting mode. Possible values:

#### Example
    <kendo:grid mode="mode">
    </kendo:grid>
