# <kendo:chart-tooltip>
A JSP tag representing Kendo Tooltip.

## Configuration Attributes


### background `String`

The background color of the tooltip. The default is determined from the series color.

#### Example
    <kendo:chart-tooltip background="background">
    </kendo:chart-tooltip>
    

### color `String`

The text color of the tooltip. The default is the same as the series labels color.

#### Example
    <kendo:chart-tooltip color="color">
    </kendo:chart-tooltip>
    

### font `String`

The tooltip font.

#### Example
    <kendo:chart-tooltip font="font">
    </kendo:chart-tooltip>
    

### format `String`

The tooltip format.

#### Example
    <kendo:chart-tooltip format="format">
    </kendo:chart-tooltip>
    

### padding `float`

The padding of the tooltip.

#### Example
    <kendo:chart-tooltip padding="padding">
    </kendo:chart-tooltip>
    

### template `String`

The tooltip template.
Template variables:

#### Example
    <kendo:chart-tooltip template="template">
    </kendo:chart-tooltip>
    

### visible `boolean`

A value indicating if the tooltip should be displayed.

#### Example
    <kendo:chart-tooltip visible="visible">
    </kendo:chart-tooltip>
