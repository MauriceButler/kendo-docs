# \<kendo:menu-items\>
A JSP tag representing Kendo Items.

#### Example
    <kendo:menu>
        <kendo:menu-items></kendo:menu-items>
    </kendo:menu>


## Child JSP Tags

### [kendo:menu-item](/api/wrappers/jsp/menu/item)

Contains items of Menu

#### Example

    <kendo:menu-items>
        <kendo:menu-item></kendo:menu-item>
    </kendo:menu-items>
